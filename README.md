# Integrador Movil 1

Aplicacion Android de gestion de contactos.

## Funcionalidades
- Login con usuario y contraseña hardcodeados
- Lista de contactos
- Busqueda por nombre
- Alta de contactos en memoria
- Boton para llamar a cada contacto
